# CSCI611 Assignment 2 - CNN (CIFAR-10)

## Requirements
- Python 3.x
- PyTorch
- torchvision
- matplotlib
- numpy

## How to run
1. Open `build_cnn.ipynb` in Jupyter Notebook / VSCode.
2. Run cells top-to-bottom (or "Run All").
3. Outputs (training/validation loss curves, test results, feature maps) will appear in the notebook.

## Files
- `build_cnn.ipynb` : training/testing + Task 2 feature visualization
- `report.pdf` : final report submitted to Canvas
